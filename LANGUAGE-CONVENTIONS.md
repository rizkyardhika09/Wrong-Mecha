## 🎬 VIDEO-RELATED CONTENT = PURE ENGLISH

**Anything that goes into or describes a published Wrong Mecha video MUST be pure English.**

### ✅ Pure English (NO mixing):

| Content | Language | Notes |
|---|---|---|
| **Video dialog (in-scene)** | English | All Riz/Vex/Kaia/Tech-Prime spoken lines |
| **Video text overlays** | English | Episode titles, captions, on-screen text |
| **YouTube title** | English | "Wrong Mecha EP X — ..." |
| **YouTube description** | English | Full description block |
| **YouTube tags** | English | All tag keywords |
| **YouTube pinned comment** | English | Including prediction polls + hype |
| **YouTube thumbnail text** | English | All text in cover/thumbnail |
| **Instagram Reels caption** | English | Cross-post copy |
| **TikTok caption** | English | Cross-post copy |
| **Story stickers/polls** | English | IG/FB stories |
| **Scene specs (DIALOG section)** | English | Per EP-spec.md files |

## ⚠️ FREQUENT MISTAKE TO AVOID

❌ **DON'T** mix languages in same paragraph for public content:
```
EPISODE 4: Brute kaiju Tier-A muncul langsung setelah cliffhanger EP 3. 
Vex masih recovery di Med Bay — Riz harus fight ALONE.
```

✅ **DO** keep pure English for public content:
```
EPISODE 4: A Tier-A Brute kaiju attacks right after EP 3 cliffhanger. 
Vex is still recovering in Med Bay — Riz has to fight ALONE.
```
