# 💡 CREDIT-SAVING TIPS untuk Generate

1. **Generate 4 variants per prompt sekaligus** (ChatGPT free tier kasih 3-4 gambar per request)
2. **Reference image** dari master asset existing — kasih ke ChatGPT sebagai "match this character" → saves moderation issues + consistency
3. **Skip ulang generate kalau ga perlu** — bisa cropping / minor edit di CapCut
4. **Lock 1 best variant per asset** → reuse di multiple scenes via Seedance Image Reference
5. **Skip optional assets** kalau bisa describe via Seedance scene prompt aja
6. **Asset reuse priority:**
   - 🔴 HIGH: Generate dulu (essential)
   - 🟡 MEDIUM: Generate kalau ada budget
   - 🟢 LOW: Skip, describe via Seedance only
