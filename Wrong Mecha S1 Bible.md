## 🌐 SERIES OVERVIEW <a id="series-overview"></a>

**Title:** Wrong Mecha

## 🌍 WORLD — PLANET AEDEN <a id="world"></a>

### Setting
- **Aeden** = native homeworld of Novarii Federation
- Rich in **AETHERCORE** crystals (planet's lifeblood)
- Cradle world being slowly destroyed by alien Hollowmen invasion (3 years ongoing)
- Atmospheric: dusk twin moons (one amber, one violet), violet-purple sky, crystal mountain landscapes

### AETHERCORE
- Sacred crystal mineral, sentient pulse, planet's lifeblood
- Glows amber (healthy) → cracked/weakening (dying)
- Source of all Novarii tech + MEK power
- Visual: hexagonal crystal cluster on ornate gold pedestal
- Hollowmen extract it to consume = killing the planet

### Aeden Locations
- **NOVARII PRIME** — underground crystal city carved into mountain (capital)
- **The Crystal Spires of Solanis** — first major Novarii city to fall (Vex's hometown, 3 years ago — lost mother + sister)
- **Hollowmen Mining Rigs** — alien extraction structures siphoning AETHERCORE
- Various war zones across rocky alien terrain

## 👥 FACTIONS <a id="factions"></a>

### 🔧 NOVARII FEDERATION (HEROES)
- Native humanoid race of Aeden
- Short scholar-warriors (1.5-2m tall — but Riz from Earth = normal human height)
- Tech-savvy, peaceful, philosophy of stewardship
- Pilot MEK mechas via cockpit (NOT cyborg bodies — pilot inside)
- Culture: crystal architecture, AETHERCORE-based tech
- Capital: NOVARII PRIME
- **Color identity:** navy blue + orange military livery (generic), white+red+blue+gold (officer), grey+gold (hero/legendary)

### 🦖 THE HOLLOWMEN (VILLAINS — formerly thought feral, Ep 7+ reveals sentience)
- Alien predator KAIJU MONSTERS (biological, NOT robots)
- Crystalline-organic chitinous bodies, bioluminescent violet veins
- Hive intelligence (more sentient than initially thought)
- Drawn to AEDEN to extract AETHERCORE
- **Tragic backstory:** Their own world was destroyed too — they are refugees seeking life
- Led by THE PRIME (SSS-Class sentient queen, full reveal Ep 9)

## 🛠️ PRODUCTION NOTES & LESSONS LEARNED <a id="production-notes"></a>

### Key Production Rules (Empirically Tested)
1. **Action scenes minimum 15s** (8s = guaranteed Seedance fail — empirically tested)
2. **2-3 image refs sweet spot** (4+ = AI averaging features into hybrid)
3. **Tech-Prime / religious-coded scenes = image refs only** (video chain triggers moderation)
4. **Voice attribution lock** for off-screen dialog (gender + intercom + mouth closed + negatives)
5. **Hard cut > difficult transitions** (climbing, complex spatial movement)
6. **Video ref OK for action continuation** (same location/characters)
7. **GPT Image 2 ALWAYS upload character master refs** (or else generic anime drift)
8. **HUD animation = CapCut overlay** (Seedance unreliable at animating text/numbers — accept face acting from Seedance + add HUD overlay manually)
9. **Cockpit master strategy** — 1 master per MEK angle, reused across all cockpit scenes
10. **Identity lock for multi-MEK scenes** — explicit naming + position separation in prompt (sky vs ground)
11. **Continuous music bed in CapCut** (avoid patah-patah audio between scenes)
12. **Color grade:** HSL boost violet/cyan/emerald + dusk warm-cool contrast
13. **30fps + Optical Flow OFF** for export (anime/AI content rule)

### Common Seedance Failure Modes + Fixes
| Failure | Fix |
|---|---|
| Characters face camera not each other | 3-layer pattern: opposing vectors + named contact + orbit camera |
| Static pose despite "stride forward" | Concrete verb + named destination + speed modifier |
| Multi-character identity blending | Explicit name + position + repeat negatives |
| Scale wrong | Comparative descriptors not numbers ("kaiju smaller than MEK") |
| Camera ignored | One move per shot |
| Prompt over 150 words | Split into multiple generations |
| Vulgar moderation block | Use softer descriptive language |
| HUD text not animating | Skip in Seedance, do CapCut overlay |
| Identity swap between MEKs | Hard naming + sky/ground separation + repeat negatives |
| Wings as shield interpreted wrong | Generate dedicated pose asset first, use as image ref |
